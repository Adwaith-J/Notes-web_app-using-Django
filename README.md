A simple CRUD (Create, Read, Update, Delete) web application built with Django to manage notes.

Features:

->Create new notes
->View all notes
->Update existing notes
->Delete notes

Tech Stack:

Backend-> Django

Database-> SQLite (default Django)

Frontend-> HTML, CSS, Bootstrap
